                         2. INSTALLATION INSTRUCTIONS

    1. Review the “JXPGenderNeutralOutfitNames License.txt” file to
       ensure you understand your rights.

    2. Extract the file “JXPGenderNeutralOutfitNames.esp” from the archive
       file “JXPGenderNeutralOutfitNames.zip” to your PC's Fallout 3 Data
       directory. The usual locations of this directory are:

           Steam Version, 64-bit Windows:
             C:\Program Files (x86)\Steam\steamapps\common\Fallout 3 goty\Data

           Steam Version, 32-bit Windows:
             C:\Program Files\Steam\steamapps\common\Fallout 3 goty\Data

           Retail Version, 64-bit Windows:
             C:\Program Files (x86)\Bethesda Softworks\Fallout 3\Data

           Retail Version, 32-bit Windows:
             C:\Program Files\Bethesda Softworks\Fallout 3\Data

    3. Additionally, if you have Point Lookout, extract the file
       “JXPGenderNeutralOutfitNamesPointLookoutCP.esp” to your PC's Fallout 3
       Data directory.

    3. Optionally, extract the files “JXPGenderNeutralOutfitNames Readme.txt”
       and “JXPGenderNeutralOutfitNames License.txt” to your PC's Fallout 3
       Data directory to retain them for future reference.

    4. Launch Fallout 3. When the Launcher window appears, select “Data Files”.
       The Fallout 3 Data Files window will appear containing a list of files.
       Click/tap the checkbox next to “JXPGenderNeutralOutfitNames.esp” and/or
       “JXPGenderNeutralOutfitNamesPointLookoutCP.esp” to enable it/them, then
       click/tap the “OK” button. The mod is now installed.

                                3. GAME CHANGES

    Changes the name of the Handyman Jumpsuit to the Handyperson Jumpsuit.

    Additionally, if you have Point Lookout, this mod changes the name of the
    Workman's Coveralls to Worker's Coveralls.

                                 4. RATIONALE

    You ain't gotta be a man to be a worker and you ain't gotta be a man to be
    handy!

                        5. UNINSTALLATION INSTRUCTIONS

    1. Delete the file(s) “JXPGenderNeutralOutfitNames.esp”, and
       “JXPGenderNeutralOutfitNamesPointLookoutCP.esp” if you have it. The mod
       is now uninstalled.
