                         2. INSTALLATION INSTRUCTIONS

    1. Review the “JXPThePittTrogItemsMissingJunkCP - License.txt” file to
       ensure you understand your rights.

    2. Extract the files “JXPThePittTrogItemsMissingJunkCP - Meshes.bsa”,
       “JXPThePittTrogItemsMissingJunkCP - Textures.bsa” and
       “JXPThePittTrogItemsMissingJunkCP.esp” from the archive file
       “JXPThePittTrogItemsMissingJunkCP.zip” to your PC's Fallout 3 Data
       directory. The usual locations of this directory are:

           Steam Version, 64-bit Windows:
             C:\Program Files (x86)\Steam\steamapps\common\Fallout 3 goty\Data

           Steam Version, 32-bit Windows:
             C:\Program Files\Steam\steamapps\common\Fallout 3 goty\Data

           Retail Version, 64-bit Windows:
             C:\Program Files (x86)\Bethesda Softworks\Fallout 3\Data

           Retail Version, 32-bit Windows:
             C:\Program Files\Bethesda Softworks\Fallout 3\Data

    3. Optionally, extract the files
       “JXPThePittTrogItemsMissingJunkCP - Readme.txt” and
       “JXPThePittTrogItemsMissingJunkCP - License.txt” to your PC's Fallout 3
       Data directory to retain them for future reference.

    4. Launch Fallout 3. When the Launcher window appears, select Data Files.
       The Fallout 3 Data Files window will appear containing a list of files.
       Click/tap the checkbox next to “JXPThePittTrogItemsMissingJunkCP.esp” to
       enable it, then click/tap the OK button. The mod is now installed.

                                3. GAME CHANGES

    This patch allows JXPThePittTrogItems and JXPMissingJunk to function
    together.

    The Yellow Pass Card will also appear on dead Trogs, along with the Blue
    Pass Card and Red Pass Card.

                        4. UNINSTALLATION INSTRUCTIONS

    1. Delete the files “JXPThePittTrogItemsMissingJunkCP - Meshes.bsa”,
       “JXPThePittTrogItemsMissingJunkCP - Textures.bsa”and
       “JXPThePittTrogItemsMissingJunkCP.esp”. The patch is now uninstalled.
