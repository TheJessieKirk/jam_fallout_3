                         2. INSTALLATION INSTRUCTIONS

    1. Review the “JXPVault101HandyBox License.txt” file to ensure you
       understand your rights.

    2. Extract the files “JXPVault101HandyBox.esp”,
       “JXPVault101HandyBox - Meshes.bsa” and
       “JXPVault101HandyBox - Textures.bsa” from the archive file
       “JXPVault101HandyBox.zip” to your PC's Fallout 3 Data directory. The
       usual locations of this directory are:

           Steam Version, 64-bit Windows:
             C:\Program Files (x86)\Steam\steamapps\common\Fallout 3 goty\Data

           Steam Version, 32-bit Windows:
             C:\Program Files\Steam\steamapps\common\Fallout 3 goty\Data

           Retail Version, 64-bit Windows:
             C:\Program Files (x86)\Bethesda Softworks\Fallout 3\Data

           Retail Version, 32-bit Windows:
             C:\Program Files\Bethesda Softworks\Fallout 3\Data

    3. Optionally, extract the files “JXPVault101HandyBox Readme.txt” and
       “JXPVault101HandyBox License.txt” to your PC's Fallout 3 Data directory
       to retain them for future reference.

    4. Launch Fallout 3. When the Launcher window appears, select Data Files.
       The Fallout 3 Data Files window will appear containing a list of files.
       Click/tap the checkbox next to “JXPVault101HandyBox.esp” to enable it,
       then click/tap the OK button. The mod is now installed.

                                3. GAME CHANGES

    This mod adds a handy empty metal box near Vault 101.

    The box is located on a rocky outcrop just East of Vault 101's entrance.

                                 4. RATIONALE

    Overencumbered with loot from your Vault 101 escape? Fear not, for now you
    have somewhere to stash that stuff so you can mule it somewhere!

                        5. UNINSTALLATION INSTRUCTIONS

    1. Delete the files “JXPVault101HandyBox.esp”,
       “JXPVault101HandyBox - Meshes.bsa” and
       “JXPVault101HandyBox - Textures.bsa”. The mod is now uninstalled.
